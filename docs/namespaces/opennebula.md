# opennebula

**Purpose:** OpenNebula infrastructure controller.

**What it does:** Runs the OpenNebula Cloud Manager (CE 7.2) components: FireEdge authentication gateway, ONE core services (VM manager, scheduler, storage), and the LLDAP integration that bridges OpenNebula user accounts with the SSO system. This namespace is responsible for provisioning and managing the underlying virtualization layer — VMs, VNETs, and image storage — for all BeckCloud workloads.
